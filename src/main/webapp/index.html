  2. Details on the Web Service

    • The service consumes and produces only JSON type
    • The principal url to access it's : localhost:8080/airlinesWebApp/ws/ + path
    • Flight Service
      PathMethodArgumentQuery Param NameQuery Param FormatDescription :
      /flight/listGETTo get all flights
      /flight/depaturesGETdate "MM/dd/yyyy hh:mm:ss"List flight by a date of depature
      /flight/arrivalsGETdate"MM/dd/yyyy hh:mm:ss"List flight by a date of arrival
      /flight/airplaneGETcodeString FormatList flight by an airplane
      /flight/numberGETnoString FormatList flight by a number of flight
      /flight/airlineGETnameStringFormatList flight by an airline
      /flight/originGETcodeString FormatList flight by airport code of depature
      /flight/destinationGETcodeString FormatList flight by airport code of destination
      /flight/newPOSTFlight ObjectAdd a new flight
      /flight/editPUTFlight ObjectEdit a flight
      /flight/deleteDELETEFlight ObjectDelete a flight
    • Airplane Service
      PathMethodArgumentQuery Param NameQuery Param FormatDescription :
      /airplane/listGETTo get all airplanes
      /airplane/modelGETmodelString FormatList airplane by model
      /airplane/serialGETcodeString FormatList flight by serial number of airplane
      /airplane/flightGETFlight ObjectList airplane by an flight
      /airplane/newPOSTAirplane ObjectAdd a new airplane
      /airplane/editPUTAirplane ObjectEdit a airplane
      /airplane/deleteDELETEAirplane ObjectDelete a airplane
