  2. Details on the Web Service

    • The service consumes and produces only JSON type
    • The principal url to access it's : localhost:8080/airlinesWebApp/ws/ + path
    • Flight Service
      Path Method Argument Query Param Name Query Param Format Description :
      /flight/list GET To get all flights
      /flight/depatures GET date "MM/dd/yyyy hh:mm:ss" List flight by a date of depature
      /flight/arrivals GET date "MM/dd/yyyy hh:mm:ss" List flight by a date of arrival
      /flight/airplane GET code String Format List flight by an airplane
      /flight/number GET no String Format List flight by a number of flight
      /flight/airline GET name StringFormat List flight by an airline
      /flight/origin GET code String Format List flight by airport code of depature
      /flight/destination GET code String Format List flight by airport code of destination
      /flight/new POST Flight Object Add a new flight
      /flight/edit PUT Flight Object Edit a flight
      /flight/delete DELETE Flight Object Delete a flight
    • Airplane Service
      Path Method Argument Query Param Name Query Param Format Description :
      /airplane/list GET To get all airplanes
      /airplane/model GET model String Format List airplane by model
      /airplane/serial GET code String Format List flight by serial number of airplane
      /airplane/flight GET Flight Object List airplane by an flight
      /airplane/new POST Airplane Object Add a new airplane
      /airplane/edit PUT Airplane Object Edit a airplane
      /airplane/delete DELETE Airplane Object Delete a airplane
